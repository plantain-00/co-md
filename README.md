# co-md
The source code of a markdown online editor with cooperation.

## develop

+ `npm i`
+ `npm run build`
+ `npm run tslint`
+ `node md.js`

## usage

+ login with Github
+ input an unique string as a room name, then share the string with others
+ click `connect`
