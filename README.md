# co-md
The source code of a markdown online editor with cooperation.

## tools and global npm packages

+ node.js >=4
+ typescript
+ tsd

## develop

+ `npm install`
+ `tsd install`
+ `tsc`
+ `node md.js`

## usage

+ login with Github
+ input an unique string as a room name, then share the string with others
+ click `connect`
